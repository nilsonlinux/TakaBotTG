## Cosa sa fare il bot?
Al momento il bot **dovrebbe** essere in grado di inviare:
* *messaggi;*
* *foto;*
* *audio;*
* *file;*
* *video;*
* *video rotondi;*
* *GIF;*
* *messaggi vocali;*
* *album con foto/video.*
